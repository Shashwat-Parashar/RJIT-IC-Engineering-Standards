# Code of Conduct

> Promoting a respectful, collaborative, and professional engineering community witihin the RJIT Innovation Club.

---

## Purpose

The purpose of this Code of Conduct is to establish clear expectations for behaviour, communication, and collaboration within the RJIT Innovation Club Engineering Standards repository.

Every contributor shares the responsibility of maintaining a respectful, inclusive, and professional environment that supports learning, innovation, and continuous improvement.


## Our Values

We strive to foster an engineering culture built upon:

- Professionalism
- Integrity
- Respect
- Collaboration
- Accountability
- Continuous Learning
- Technical Excellence


## Expected Behaviour

Contributors are expected to:

- Treat all members with courtesy and respect.
- Encourage constructive technical discussions.
- Provide helpful and professional feedback.
- Respect differing technical opinions.
- Support collaboration across multidisciplinary teams.
- Share knowledge openly and responsibly.
- Follow the repository's engineering documentation standards.


## Unacceptable Behaviour

The following behaviours are not acceptable:

- Harassment, discrimination, or personal attacks.
- Offensive or abusive language.
- Intentional disruption of discussions or collaboration.
- Plagiarism or misrepresentation of another person's work.
- Knowingly submitting misleading or false technical information.
- Any behaviour that creates an unsafe or unprofessional environment.


## Professional Engineering Conduct

Engineering documentation should always be:

- Accurate
- Honest
- Traceable
- Well documented
- Technically justified

Contributors should prioritize evidence-based engineering decisions and clearly document assumptions, limitations, and design choices whenever applicable.


## Collaboration

Engineering is a collaborative discipline.

Contributors are encouraged to:

- Work respectfully with fellow members.
- Welcome constructive reviews.
- Share technical knowledge.
- Help improve existing documentation.
- Support new contributors.


## Reporting Concerns

If you experience or witness behaviour that violates this Code of Conduct, please report it to the repository maintainers or the RJIT Innovation Club faculty mentor.

Reports will be reviewed fairly, respectfully, and confidentially whenever possible.


## Enforcement

Repository maintainers reserve the right to edit, reject, or remove contributions that do not comply with this Code of Conduct.

Repeated or serious violations may result in restrictions on future contributions.


## Scope

This Code of Conduct applies to:

- Repository discussions
- Documentation reviews
- Pull Requests
- Issues
- Engineering documentation
- Collaborative activities related to this repository


## Contact

Questions regarding this Code of Conduct may be directed to the maintainers of the RJIT Innovation Club Engineering Standards repository.
