# RJIT Innovation Club Engineering Standards

> Engineering documentation standards, templates, and engineering workflow developed for the RJIT Innovation Club.

---

## About

The **RJIT Innovation Club Engineering Standards (RJIT-IC-EDS)** repository establishes a standardized framework for engineering documentation, project management, and technical reporting within the RJIT Innovation Club.

Its purpose is to ensure consistency, professionalism, traceability, and knowledge preservation across all club projects.

This repository serves as the central source for:

- Engineering Documentation Standards
- Project Proposal Templates
- Design Documentation
- Engineering Reports
- CAD Documentation Standards
- Testing Procedures
- Software Documentation
- Project Examples

---

## Objectives

The repository aims to:

- Standardize engineering documentation.
- Improve collaboration between multidisciplinary teams.
- Preserve engineering knowledge across student batches.
- Support professional engineering practices.
- Create reusable documentation templates.
- Build a long-term technical knowledge base for the Innovation Club.
